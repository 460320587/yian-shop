## 快速定位

### 按角色

**后端开发（新入职）**：
1. 读 `01-架构总纲.md`（理解整体架构）
2. 读 `03-工程规范与分层.md`（理解代码组织方式）
3. 读 `04-数据库设计.md`（理解数据模型）
4. 根据负责的模块读对应的业务卷

**前端开发**：
1. 读 `01-架构总纲.md`（理解架构边界）
2. 在业务卷中搜索 API 路由和 Resource 定义

**DBA/运维**：
1. 读 `02-技术栈与基础设施.md`
2. 读 `04-数据库设计.md`
3. 读 `06-核心业务-卷二.md` 中的部署运维章节

**测试/QA**：
1. 读 `95-审查报告与FR映射.md`（FR覆盖度和映射关系）
2. 在业务卷中搜索接口契约
