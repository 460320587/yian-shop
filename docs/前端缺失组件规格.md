## 1. AppCaptcha 验证码组件

### 使用场景
- 用户注册、登录、找回密码时的人机验证
- 高频操作（如频繁发送短信）的二次验证

### Props

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `type` | `'image' \| 'slide' \| 'click'` | 否 | `'image'` | 验证码类型：图片/滑块/点选 |
| `width` | `number` | 否 | `120` | 验证码图片宽度(px) |
| `height` | `number` | 否 | `40` | 验证码图片高度(px) |
| `length` | `number` | 否 | `4` | 验证码字符长度 |
| `caseSensitive` | `boolean` | 否 | `false` | 是否区分大小写 |
| `refreshable` | `boolean` | 否 | `true` | 是否可刷新 |
| `apiEndpoint` | `string` | 是 | - | 验证码获取API地址 |
| `verifyEndpoint` | `string` | 是 | - | 验证码校验API地址 |
| `errorCount` | `number` | 否 | `0` | 当前错误次数（触发加强验证） |

### Events

| Event | 参数 | 说明 |
|-------|------|------|
| `verify` | `(valid: boolean, token: string)` | 验证结果回调 |
| `refresh` | `()` | 刷新验证码 |
| `error` | `(msg: string)` | 验证失败/网络错误 |

### Slots

| Slot | 说明 |
|------|------|
| `loading` | 加载中自定义内容 |
| `error` | 加载失败自定义内容 |

### TypeScript Interface

```typescript
// src/components/common/AppCaptcha/types.ts
export interface CaptchaConfig {
  type: 'image' | 'slide' | 'click';
  width: number;
  height: number;
  length: number;
  caseSensitive: boolean;
  refreshable: boolean;
  apiEndpoint: string;
  verifyEndpoint: string;
  errorCount: number;
}

export interface CaptchaResponse {
  captchaId: string;
  imageBase64: string;        // 图片验证码
  slideImage?: string;        // 滑块底图
  slideBlock?: string;        // 滑块图片
  clickPoints?: ClickPoint[]; // 点选坐标提示
}

export interface ClickPoint {
  x: number;
  y: number;
}

export interface VerifyPayload {
  captchaId: string;
  code?: string;              // 图片验证码输入值
  track?: SlideTrack;         // 滑块轨迹
  points?: ClickPoint[];      // 点选坐标
}

export interface SlideTrack {
  startX: number;
  startY: number;
  track: Array<{ x: number; y: number; t: number }>;
}
```

### 使用示例

```vue
<template>
  <AppCaptcha
    type="image"
    :width="120"
    :height="40"
    api-endpoint="/api/captcha/generate"
    verify-endpoint="/api/captcha/verify"
    @verify="onCaptchaVerify"
  />
</template>

<script setup lang="ts">
import AppCaptcha from '@/components/common/AppCaptcha/index.vue';

const onCaptchaVerify = (valid: boolean, token: string) => {
  if (valid) {
    formData.captchaToken = token;
  }
};
</script>
```

## 2. AppAddressPicker 地址选择器

### 使用场景
- 下单时选择收货地址
- 个人中心添加/编辑地址
- 运费模板配置时选择区域

### Props

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `modelValue` | `AddressItem` | 否 | `null` | 当前选中地址(v-model) |
| `level` | `number` | 否 | `3` | 选择层级 1省/2省市/3省市区/4省市区街道 |
| `showDetail` | `boolean` | 否 | `true` | 是否显示详细地址输入 |
| `showLabel` | `boolean` | 否 | `true` | 是否显示地址标签选择 |
| `showZipCode` | `boolean` | 否 | `false` | 是否显示邮编输入 |
| `labels` | `string[]` | 否 | `['家','公司','工厂']` | 可选标签列表 |
| `disabled` | `boolean` | 否 | `false` | 是否禁用 |
| `saveable` | `boolean` | 否 | `true` | 是否允许保存到地址簿 |
| `addressList` | `AddressItem[]` | 否 | `[]` | 已保存地址列表 |

### Events

| Event | 参数 | 说明 |
|-------|------|------|
| `update:modelValue` | `(address: AddressItem)` | 地址变更 |
| `save` | `(address: AddressItem)` | 保存到地址簿 |
| `select` | `(address: AddressItem)` | 从列表选择 |
| `delete` | `(id: number)` | 删除地址 |

### Slots

| Slot | 说明 |
|------|------|
| `header` | 顶部自定义内容 |
| `footer` | 底部自定义内容 |

### TypeScript Interface

```typescript
// src/components/common/AppAddressPicker/types.ts
export interface Region {
  code: string;
  name: string;
  children?: Region[];
}

export interface AddressItem {
  id?: number;

  cityCode: string;
  district: string;
  districtCode: string;

  zipCode?: string;

  label?: string;

  modelValue: AddressItem | null;
  level: 1 | 2 | 3 | 4;

  disabled: boolean;
  saveable: boolean;

```

### 使用示例

```vue
<template>
  <AppAddressPicker
    v-model="selectedAddress"
    :level="3"
    :saveable="true"
    :address-list="savedAddresses"
    @save="onSaveAddress"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue';
import type { AddressItem } from '@/components/common/AppAddressPicker/types';

const selectedAddress = ref<AddressItem | null>(null);
const savedAddresses = ref<AddressItem[]>([]);

const onSaveAddress = (addr: AddressItem) => {
  savedAddresses.value.push(addr);
};
</script>
```

## 3. AppImageCropper 图片裁剪器

### 使用场景
- 上传头像时的裁剪
- 上传设计文件时的尺寸校验和裁剪
- 商品图片上传时的固定比例裁剪

### Props

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `src` | `string` | 是 | - | 原图URL/base64 |
| `aspectRatio` | `number` | 否 | `NaN` | 固定裁剪比例 16/9=1.78, 4/3=1.33, 1=正方形 |
| `minWidth` | `number` | 否 | `100` | 最小裁剪宽度(px) |
| `minHeight` | `number` | 否 | `100` | 最小裁剪高度(px) |
| `maxWidth` | `number` | 否 | `4096` | 最大裁剪宽度(px) |
| `maxHeight` | `number` | 否 | `4096` | 最大裁剪高度(px) |
| `outputType` | `'blob' \| 'base64' \| 'file'` | 否 | `'blob'` | 输出格式 |
| `outputQuality` | `number` | 否 | `0.92` | 输出质量 0-1 |
| `outputMaxWidth` | `number` | 否 | `1920` | 输出最大宽度 |
| `outputMaxHeight` | `number` | 否 | `1920` | 输出最大高度 |
| `viewMode` | `0 \| 1 \| 2 \| 3` | 否 | `1` | 裁剪框限制模式 |
| `guides` | `boolean` | 否 | `true` | 是否显示辅助线 |
| `center` | `boolean` | 否 | `true` | 是否显示中心指示器 |
| `modal` | `boolean` | 否 | `true` | 是否显示模态背景 |
| `title` | `string` | 否 | `'图片裁剪'` | 弹窗标题 |

### Events

| Event | 参数 | 说明 |
|-------|------|------|
| `crop` | `(result: CropResult)` | 裁剪完成 |
| `cancel` | `()` | 取消裁剪 |
| `error` | `(msg: string)` | 图片加载错误 |
| `ready` | `()` | 裁剪器就绪 |

### Methods (Expose)

| Method | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| `getCropData` | `()` | `CropResult` | 获取裁剪结果 |
| `rotate` | `(deg: number)` | `void` | 旋转图片 |
| `zoom` | `(ratio: number)` | `void` | 缩放图片 |
| `reset` | `()` | `void` | 重置裁剪框 |

### TypeScript Interface

```typescript
// src/components/common/AppImageCropper/types.ts

export interface ImageCropperProps {
  src: string;
```

### 使用示例

```vue
<template>
  <AppImageCropper
    ref="cropperRef"
    :src="uploadedImage"
    :aspect-ratio="4/3"

  />

<script setup lang="ts">

import type { CropResult } from '@/components/common/AppImageCropper/types';

const uploadedImage = ref('');

const onCrop = (result: CropResult) => {
  const formData = new FormData();
  formData.append('file', result.blob!);
  // 上传裁剪后的图片
};
</script>
```

## 4. AppPdfViewer PDF预览器

### 使用场景
- 设计文件PDF预览（上传后预览）
- 订单详情中查看工艺单/合同PDF
- 帮助文档PDF在线查看

### Props

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `src` | `string` | 是 | - | PDF文件URL/base64 |
| `page` | `number` | 否 | `1` | 当前页码(v-model) |
| `scale` | `number` | 否 | `1.0` | 缩放比例 |
| `fit` | `'width' \| 'height' \| 'page' \| 'none'` | 否 | `'width'` | 自适应模式 |
| `toolbar` | `boolean` | 否 | `true` | 是否显示工具栏 |
| `printable` | `boolean` | 否 | `true` | 是否允许打印 |
| `downloadable` | `boolean` | 否 | `true` | 是否允许下载 |
| `searchable` | `boolean` | 否 | `false` | 是否启用搜索 |
| `rotation` | `0 \| 90 \| 180 \| 270` | 否 | `0` | 旋转角度 |
| `loadingText` | `string` | 否 | `'PDF加载中...'` | 加载提示 |

### Events

| Event | 参数 | 说明 |
|-------|------|------|
| `update:page` | `(page: number)` | 页码变更 |
| `update:scale` | `(scale: number)` | 缩放变更 |
| `loaded` | `(info: PdfInfo)` | PDF加载完成 |
| `error` | `(msg: string)` | 加载错误 |
| `pageChange` | `(page: number, total: number)` | 翻页事件 |

### Slots

| Slot | 说明 |
|------|------|
| `toolbar-left` | 工具栏左侧自定义 |
| `toolbar-right` | 工具栏右侧自定义 |
| `loading` | 自定义加载状态 |
| `error` | 自定义错误状态 |

### TypeScript Interface


## 5. App3dPreviewer 3D预览器

### 使用场景
- 包装盒/袋类产品的3D效果预览
- 商品详情页展示3D模型
- 客户定制效果3D实时预览

### Props

| Prop | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `modelUrl` | `string` | 是 | - | 3D模型文件URL (.glb/.gltf/.obj) |
| `textureUrl` | `string` | 否 | `''` | 贴图URL |
| `autoRotate` | `boolean` | 否 | `true` | 是否自动旋转 |
| `rotateSpeed` | `number` | 否 | `1.0` | 自动旋转速度 |
| `backgroundColor` | `string` | 否 | `'#f5f5f5'` | 背景色 |
| `width` | `number \| string` | 否 | `'100%'` | 宽度 |
| `height` | `number \| string` | 否 | `400` | 高度(px) |
| `cameraPosition` | `[number, number, number]` | 否 | `[5,5,5]` | 初始相机位置 [x,y,z] |
| `lightIntensity` | `number` | 否 | `1.0` | 灯光强度 |
| `shadows` | `boolean` | 否 | `true` | 是否显示阴影 |
| `grid` | `boolean` | 否 | `false` | 是否显示网格 |
| `controls` | `'orbit' \| 'trackball' \| 'none'` | 否 | `'orbit'` | 控制模式 |
| `annotations` | `Annotation[]` | 否 | `[]` | 3D标注点 |

### Events

| Event | 参数 | 说明 |
|-------|------|------|
| `load` | `()` | 模型加载完成 |
| `error` | `(msg: string)` | 模型加载错误 |
| `click` | `(intersection: Intersection)` | 3D对象点击 |
| `hover` | `(intersection: Intersection \| null)` | 鼠标悬停 |
| `render` | `(info: RenderInfo)` | 每帧渲染回调 |

### Slots

| Slot | 说明 |
|------|------|
| `loading` | 自定义加载状态 |
| `overlay` | 3D场景上叠加UI |
| `controls` | 自定义控制面板 |

### TypeScript Interface

```typescript
// src/components/common/App3dPreviewer/types.ts
import type { Vector3 } from 'three';

export interface Annotation {
  id: string;
  position: [number, number, number];
  title: string;
  description?: string;
  color?: string;
}

export interface Intersection {
  point: Vector3;
  objectName: string;
  distance: number;
  uv?: { x: number; y: number };
}

export interface RenderInfo {
  fps: number;
  frameTime: number;
  triangles: number;
}

export interface PreviewerProps {
  modelUrl: string;
  textureUrl: string;
  autoRotate: boolean;
  rotateSpeed: number;
  backgroundColor: string;
  width: number | string;
  height: number | string;
  cameraPosition: [number, number, number];
  lightIntensity: number;
  shadows: boolean;
  grid: boolean;
  controls: 'orbit' | 'trackball' | 'none';
  annotations: Annotation[];
}

// 支持的3D模型格式
export type ModelFormat = 'glb' | 'gltf' | 'obj' | 'fbx' | 'stl';
```

### 使用示例

```vue
<template>
  <App3dPreviewer
    model-url="/models/box.glb"
    :texture-url="uploadedDesignTexture"
    :auto-rotate="true"

    :shadows="true"
    :annotations="productAnnotations"
    @click="on3dClick"
  >
    <template #overlay>
      <div class="annotation-tooltip" v-if="hoveredAnnotation">

      </div>
    </template>

const productAnnotations: Annotation[] = [
  { id: '1', position: [1, 0.5, 0], title: '覆膜区域' },
  { id: '2', position: [-1, 0, 0.5], title: '烫金位置' },
];

const on3dClick = (intersection: Intersection) => {
  console.log('点击了:', intersection.objectName);
};
</script>
```
